# Session Log: 2026-03-28 — From Structure to Signal

## The Analogy: A Mail Sorting Office

The neural graph is a mail sorting office. Clusters are sorters. Edges are
conveyor belts. Input arrives as mail (CLIP vectors). The office must organize
itself to sort mail correctly and eventually describe what it's sorting.

---

## Phase 1: Multi-Round Attention (merged from branch)

**What we built:** Instead of one pass through the office, mail bounces between
sorters multiple times. Each round, sorters attend to what others found. They
converge when no one is surprised anymore.

```
Before:  mail → sorter A → done
After:   mail → sorter A → "hmm" → check B → "aha" → check A again → done
```

**Metrics at step 41K:**
```
spatial:     0.579  (new high, up from 0.54)
communities: 4
avg_sim:     0.076
```

**Verdict:** ✓ Worked. Spatial improved. Merged to main.

## Phase 2: Sequential Curriculum + Temporal Co-firing

**What we built:** Instead of random flashcards, show the baby 16 images of the
same category, then switch. Like a teacher showing a chapter on dogs before
moving to cats. Also track which sorters fire in sequence (temporal co-firing).

```
Before:  dog, bus, cat, dog, horse, bus, cat...  (shuffled deck)
After:   dog, dog, dog...×16, cat, cat, cat...×16  (chapters)
```

**Metrics at step 64K (on existing mature graph):**
```
spatial:     0.63   (↑ from 0.579)
communities: 16     (↑ from 4)
avg_sim:     0.076  (held)
new cats:    49 tracked (up from 13)
```

**Verdict:** ✓ Big win. Communities exploded. All categories now visible.

## Phase 3: Learned Projection (Global)

**Hypothesis:** The CLIP embeddings are a "frosted window" — sorters can feel
the shape of mail but can't read the addresses. A learned 512×512 rotation
matrix would clear the frost.

```
CLIP (512) → GLOBAL ROTATION (512×512) → graph
```

**Attempt 1: With momentum**
```
@18K: spatial 0.415, best +0.089, worst -0.261  ← CRASHED. Momentum +
      sequential episodes = whipsaw. 16 steps pushing dog-ward, then
      violent reversal at cat boundary.
```

**Attempt 2: No momentum, scaled LR**
```
@32K: spatial 0.197, best +0.040, worst -0.128  ← Too slow. Delta norm
      only 1.77. Projection barely moving.
```

**Attempt 3: 5x LR**
```
@55K: spatial 0.435, best +0.108, worst -0.083  ← Helped some categories,
      hurt others. Chair flipped positive, sheep went negative.
```

**Verdict:** ✗ A single rotation can't serve 49 categories. Rotating space to
help "chair" misaligns "sheep."

---

## Phase 4: Per-Cluster Lens

**Hypothesis:** Instead of one rotation for everyone, each sorter gets custom
glasses. A (512,) bias vector per cluster corrects the input locally.

```
Before:  ALL sorters see same rotated mail
After:   each sorter adjusts their own view: normalize(x + lens_i)
```

**Attempt 1: Global error signal**
```
@25K: spatial 0.229, best +0.156, worst -0.075  ← Same negatives. The
      "error" each lens trains on is the GLOBAL gap, not per-cluster.
      A dog-specialist gets bus-corrections in its lens.
```

**Attempt 2: Per-cluster error signal**
```
@25K: spatial 0.229, best +0.160, worst -0.075  ← Same result. Negatives
      from different source than we thought.
```

**Verdict:** ✗ Lens adds noise to early training, hurting spatial formation.
More importantly: negatives aren't caused by the lens OR the projection.

---

## Phase 5: Root Cause Discovery

**The breakthrough:** Negatives appear in EVERY sequential curriculum run,
lens or not. The old random-curriculum baseline had ZERO negatives.

```
Random curriculum:     avg_sim 0.05-0.07, no negatives, bland
Sequential curriculum: avg_sim 0.18 best, -0.08 worst, spiky
```

**Root cause: Activation buffer bleed.** The buffer carries category A's
signal into category B's episode. 16 dogs saturate the buffer, then cats
arrive and the buffer makes clusters output dog-like responses for cat input.

**Fix: Palate cleanser.** Zero the buffer at episode boundaries. Like a
sommelier's sorbet between wine flights.

```
@50K (no lens, with cleanser): spatial 0.591, best +0.18, worst -0.03
```

Negatives shrinking toward zero. Spatial best-ever for a no-lens run.

---

## Phase 6: The Key Insight — Stop Resetting

**We kept resetting the model every time we saw negatives.** Six resets
this session. But the FIRST sequential run (added to a 70K-step mature
graph) had negatives at 10K that resolved by 40K:

```
First sequential run:
  Step 10K:  tennis -0.21  ← thrashing
  Step 41K:  ALL POSITIVE  ← resolved
  Step 64K:  spatial 0.63, 28 communities  ← best ever
```

**The negatives are early-training thrashing, not a bug.** A fresh 4-cluster
graph can't handle 49 categories. By 40-60K, with 500+ clusters, every
category finds its home. We just never let any run mature long enough to
see this.

**Analogy:** Teaching a child to swim. We kept pulling them out of the pool
to change their goggles. The child who stayed in learned to swim by minute 60.

## Phase 7: Episodic Memory

**What we built:** Store high-error and growth-event experiences. Replay 8
memories per batch, prioritizing worst-performing categories.

```
see dogs → learn → STORE best → see cats → REPLAY dogs → consolidate
```

- 2000 memory capacity in SQLite
- Category-balanced replay weighted by inverse performance
- Rate-limited storage (top-25% error or growth events)

**Status:** Active in current run. 600+ memories stored. Contribution TBD —
need to reach 80K+ to evaluate against non-memory baseline.
