## Architecture Changes

### Two-Process Architecture
```
Process 1: train_worker.py (owns brain, MPS GPU, SQLite)
  Pipeline per batch:
    prepare → compute → distill → categories → growth →
    cofiring → reasoning → decoder → checkpoint → publish

Process 2: main.py (HTTP server, always responsive)
  Reads shared_state.json for /status, /metrics, /dashboard
  Sends commands via train_command.json
  WebSocket sends graph snapshots + deltas
```

### Performance Optimizations
```
Forward pass:  92.88ms → 14.04ms (6.6x)
  - Eliminate .item() GPU sync calls (5.5x)
  - @torch.no_grad() (eliminates autograd tracking)
  - Skip dormant neurons in SENSE matmul (36% less compute)
  - Think (message passing) every 20 steps not 5
  - Hebbian edges every 10 steps not every step
  - Pre-sense weight caching across batch items
```

### Bugs Fixed (12)
1. Projector IndexError (1,000+/session) — SVD with <3 dims
2. _ClusterView.cluster_type missing — frontend crash
3. Event loop blocking — batch in thread pool
4. SQLite thread safety — check_same_thread=False
5. Distill counter stuck at 1 — increment outside gate
6. Garbage text vectors — use CLIP until native converges
7. Reasoning clobbers brain state — save/restore buffer
8. Cofiring buffer 50K → 5K — prevented GIL stalls
9. SQLite database locked — thread-local connections + busy_timeout
10. Modulo alignment — fuzzy thresholds for item-count gating
11. MPS generator device mismatch — create on CPU then move
12. GPU sync overhead — batch .cpu()/.tolist() instead of per-neuron .item()

## Brain Capabilities

### What It Can Do
```
"A man surfing on a wave"              → "wave man beach"
"A red and yellow bus on the road"     → "bus red road yellow"
"Dogs playing in a grassy field"       → "dog garden grass beach"
"A room with chair, couch, mirror"     → "chair room window"
"ann has a red cat"                    → "red cat tomorrow"
```

### What It Can't Do Yet
```
"what does dan have?" (answer: bear)   → "cow dan" (wrong retrieval)
"what color is the shoe?" (orange)     → "table shoe purple" (wrong color)
Multi-step reasoning, image generation, raw pixel processing
```
