# Concept Brain v3 — Vision

## One sentence
A self-organizing concept graph that stores knowledge in one shot, reasons through typed connections, develops its own encoding, and generates output from activation patterns — all in one unified structure with fractal hierarchy and emergent specialization.

## The key insight
Not 8 modules bolted together. ONE substance with multiple views. Like a body where bones, muscles, nerves, blood all occupy the same space, superimposed, integrated.

Every concept node simultaneously IS:
- A memory (stored fact)
- A knowledge vertex (connected to related concepts)
- A cluster member (part of an emergent region)
- An activation target (can be "lit up" by a query)
- A building block (can be combined with others)
- A walk step (can be discovered through random exploration)
- An output source (can generate text/image/sound)

ONE node. MANY functions. Not separate systems.

## Core properties
- **Superimposed**: all operations on the same graph, not separate modules
- **Emergent regions**: clusters form from data, not from pre-designed architecture
- **Hot-swappable**: transplant regions between brains (train separately, merge)
- **One-shot storage**: write a concept in one step, not 10,000 gradient updates
- **Cross-modal**: one node handles text + image + sound + abstract
- **Self-encoding**: space defined by relationships, not inherited from CLIP
- **Temporal**: sequence is just typed edges (then, before, after)
- **Compositional**: edge types guide how to combine concepts (purple + dog)
- **Fractal**: concepts → clusters → regions → systems (same pattern at every scale)
- **Creative**: random walks discover unexpected connections (analogies, insights)

## What it replaces
- BrainV2 (10K neurons, FF learning, REFLECT) → Concept Graph (one-shot, typed edges, inference)
- Separate encoder/decoder modules → Unified parse/generate operations on same graph
- CLIP-dependent encoding → Self-developing concept space
- Gradient learning for facts → One-shot storage + connection
- Flat neuron network → Fractal hierarchical concept graph
