The Brain
Four building blocks, one growing structure.
⚡
Node (neuron)

512 numbers = its personality. Fires via tanh(dot(weights, input)). After each update, weights are L2-normalized so they can't explode.

🧩
Cluster (concept)

8 nodes that work together. Has an "identity" = the average direction all its nodes point. Types: Integration, Transformation, Routing, Arbitration.

🔗
Edge (connection)

Links two clusters. Strength 0–1. Updates via Oja's rule: self-normalizing, edges settle at the true correlation instead of all drifting to 1.0.

🌳
Quadtree (spatial index)

Organizes clusters in 2D space. Each tile is 64×64. Splits when internal variance gets high. O(log n) lookup instead of scanning everything.

BUD naming — how clusters form a family tree 
c_00  ← original cluster
  ├── c_00a  ← first split, child A
  │     ├── c_00aa  ← second split of A
  │     └── c_00ab
  └── c_00b  ← first split, child B

depth = number of suffix letters (c_00=0, c_00a=1, c_00ab=2)
parent = ID with last letter removed
Note: BUD removes the parent from the graph
Learning
Each neuron only knows one thing: did the last signal feel good or bad?
Forward-Forward learning. No backpropagation. Each neuron adjusts itself locally.
Image/text
CLIP → 512 numbers
Brain processes it
Compare to teacher
✓ or ✗
✓
Positive

Weights shift toward the input. "I liked that — more things like this."

✗
Negative

Weights shift away. "That was noise — less like that."

The actual math 
Positive: weights += lr × plasticity × |activation| × input × (1 − activation²)
Negative: weights −= lr × plasticity × |activation| × input × (1 − activation²)
Then:     weights = L2_normalize(weights)   ← prevents saturation

The (1 − activation²) term is the tanh derivative. It naturally reduces updates when the neuron is already confident (near ±1), and allows larger updates when uncertain (near 0).

Stage-aware learning rate
StageMultiplierWhy
Stage 0 (sensory)1.0×Full speed — building initial structure
Stage 1 (association)0.5×Refining — don't overwrite what's learned
Stage 2+ (concept)0.1×Fine-tuning — subtle adjustments only

Base rate decays exponentially: 0.01 × exp(−0.0003 × step). At step 7000, Stage 2: effective LR ≈ 0.0002.

In batch mode (32 samples), LR is divided by batch size so total weight movement matches a single step.

How positive/negative is determined
StageMethod
Stage 0Every 3rd step is negative (deterministic schedule)
Stage 1+Cosine similarity between model output and teacher answer, compared against adaptive median threshold over last 25 steps. If positive rate drops below 40%, threshold lowers.

The Problem
It learns, but it doesn't specialize.
All 334 clusters fire for almost everything. They should form distinct islands. Instead: one uniform blob.
What we have: the blob

Every cluster responds to every input roughly equally. No "dog region" or "car region." Just a uniform mass.

What we want: islands

Dog detectors, car detectors, nature integrators. Distinct regions with sparse bridges between them.

🎯
One-direction learning

Each cluster points one way. Learns "average of dogs" — not "the category of dogs." Can't represent groups.

🪞
Clone army

Every cluster is identical. Same rules, same shape. Like a football team where everyone plays midfielder.

🩹
Band-aid growth

Growth only fires when something breaks. Never scouts ahead. Never builds toward unknown territory.

🎲
Random curriculum

Training samples are random. Brain never gets targeted practice on its weaknesses.

🔗
Dumb connections

Edges are just a number. Can't say "only fire in this context." Dog→frisbee always fires.

🐟
Goldfish memory

Each step is independent. Forgets everything between inputs. Can't learn sequences.

The failure equilibria — why the blob is stable

The Blob: All clusters converge to similar directions → all co-fire → all edges strengthen → inhibition applies equally → no differentiation. Stable because any specialist gets fewer activations → lower Hebbian reinforcement → gets pruned. Generalists survive.

The Fossil: After memorizing the curriculum, growth stops. No new BUDs (low bimodality). No new INSERTs (small residuals). Graph fossilizes. New concepts can't form.

Both are stable because the mechanism rewards convergence: FF → mean direction, Hebbian → strengthen co-firing, Resonance → activate similar, Growth → only split if confused.

The Plan
Six changes that turn the blob into a real brain.
One at a time. Verify each before moving on. Safest changes first.
1. Adversarial Curriculum
Track what the brain is bad at. Show it more of that. Add hard negatives — things that look similar but aren't (dog vs wolf). Target the weakness.
2. Memory Buffer
Short-term memory: "I just saw a dog" lingers for a few steps. "Leash" and "park" become more likely. Context makes everything smarter.
3. Multi-Prototype Clusters
3 directions per cluster instead of 1. A cluster can mean "dogs AND cats AND birds" = animals. Categories instead of single points.
4. Curiosity Growth
When nothing recognizes an input, create a new cluster for it. Build toward the unknown. Explore, don't just patch.
5. Cluster Roles
Three jobs: Detectors (spot patterns), Integrators (combine signals), Predictors (anticipate). Different roles → specialization.
6. Smart Edges
Connections that know when to fire. Excitatory, inhibitory, modulatory. "Dog→frisbee" only in outdoor context.

Risks to watch
  • Interaction explosion — 6 new mechanisms × 6 existing = 36 interactions. Implement one phase at a time.
  • Hyperparameter sensitivity — 8 new parameters. Start with defaults, only tune what matters.
  • Catastrophic forgetting — New mechanisms shift the training distribution. Ramp up gradually over 500 steps.
  • Buffer hallucination — Memory too strong → model sees what it expects, not what's there. Monitor output similarity to current vs previous input.
The Bet

The learning rule works. The growth operations work. What's missing is the game design — the rules that incentivize clusters to specialize, edges to carry meaning, and growth to explore instead of patch. We're not rebuilding the engine. We're changing the game it plays.
